# Skill: 50-word paragraph generator

This skill instructs Claude code to write a 50-word paragraph on a given topic name.

## Instructions

When invoked, generate exactly one paragraph with 50 words.

- Use the provided topic name as the subject.
- Keep the paragraph cohesive and on-topic.
- Do not include headings, lists, or extra formatting.
- Ensure the output contains exactly 50 words.

## Example

Input: "sustainability"
Output: A concise paragraph about sustainability in exactly fifty words.
